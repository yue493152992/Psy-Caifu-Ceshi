### 🎯 设计目标
1. **吸引用户** - 3秒内让用户理解产品价值
2. **激发兴趣** - 引发用户对自我认知的好奇
3. **引导行动** - 促使用户点击"开始测试"

### 📊 成功指标
- 页面跳出率 < 50%
- 测试转化率 > 60%
- 平均停留时间 > 20秒
